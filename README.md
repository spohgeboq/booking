# Система Онлайн-Записи (Online Booking System)

Данный проект представляет собой комплексную систему для онлайн-записи клиентов, состоящую из бэкенда и двух фронтенд-приложений.

## Структура проекта

- **`backend/`**: Node.js сервер, обеспечивающий работу API, интеграцию с базой данных (Supabase) и уведомления в Telegram.
- **`frontend/`**: Панель администратора для управления записями, мастерами и услугами.
- **`booking-frontend/`**: Клиентский интерфейс для выбора услуг и записи на прием.

## Быстрый старт

### 1. Настройка переменных окружения

В каждой из папок (`backend`, `frontend`, `booking-frontend`) создайте файл `.env` на основе предоставленных файлов `.env.example`.

### 2. Установка зависимостей

Перейдите в каждую папку и выполните установку:
```bash
cd backend && npm install
cd ../frontend && npm install
cd ../booking-frontend && npm install
```

### 3. Запуск в режиме разработки

Для каждой части проекта:
```bash
npm run dev
```

## Технологии

- **Frontend**: React, Vite, TypeScript, TailwindCSS.
- **Backend**: Node.js, Express.
- **Database**: Supabase (PostgreSQL).
- **Notifications**: Telegram Bot API.

## Подготовка к Git

Проект настроен с использованием единого репозитория. Все временные файлы (`node_modules`), папки сборки (`dist`, `build`) и локальные настройки (`.env`) добавлены в `.gitignore`.
